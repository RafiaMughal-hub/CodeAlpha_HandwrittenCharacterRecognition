# Handwritten Character Recognition Using CNN

## Overview

This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits from image data. The model is trained on the MNIST dataset and achieves high accuracy in digit classification.

## Objective

To develop a deep learning model capable of identifying handwritten digits from images.

## Dataset

MNIST Handwritten Digit Dataset

* 60,000 Training Images
* 10,000 Testing Images
* 10 Digit Classes (0–9)

## Technologies Used

* Python
* TensorFlow
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Deep Learning Model

Convolutional Neural Network (CNN)

Layers Used:

* Conv2D
* MaxPooling2D
* Flatten
* Dense
* Dropout

## Evaluation Metrics

* Test Accuracy
* Classification Report
* Confusion Matrix

## Results

The CNN model achieved high classification accuracy and successfully recognized handwritten digits with excellent performance.
